# rock-paper-scissors
Rock Paper Scissors Game

A simple Rock Paper Scissors game built using HTML and JavaScript. The game allows the user to play against the computer through browser prompts and displays the result using alert messages.

Features
- User selects Rock, Paper, or Scissors.
- Computer randomly generates its choice.
- Determines the winner based on game rules.
- Displays both choices and the result.
- Beginner-friendly JavaScript project.

  
Technologies Used
- HTML
- JavaScript
  
Future Improvements
- Add buttons instead of prompt inputs
- Create a modern user interface with CSS
- Keep score across multiple rounds
- Add animations and sound effects
- Make the game mobile-friendly
